## 开发环境

```bash
# 创建集群，若加上 -d 参数表示后台运行。

$ ./lnmp-docker clusterkit-mysql-up [-d]

# 销毁集群，若加上 -v 参数会销毁数据卷。

$ ./lnmp-docker clusterkit-mysql-down [-v]
```

### Swarm mode

```bash
# 创建集群

$ ./lnmp-docker clusterkit-mysql-deploy

# 销毁集群

$ ./lnmp-docker clusterkit-mysql-remove
```

# PHP 连接集群

主服务器 `root` 账号

从服务器 `node` 账号（拥有只读权限）

    'read' => [
        'host' => '192.168.1.1',
        'port' => '3306',
    ],
    'write' => [
        'host' => '196.168.1.2',
        'port' => '3306'
    ],
    'sticky'    => true,
    'driver'    => 'mysql',
    'database'  => 'database',
    'username'  => 'root',
    'password'  => '',
    'charset' => 'utf8mb4',
    'collation' => 'utf8mb4_unicode_ci',
    'prefix'    => '',
],
